# Notes App (Firebase)

A real-time notes application built with React and Firebase.  
Features authentication, CRUD operations, real-time updates, and a responsive UI.

## Features
- User authentication (Email/Password)
- Create, edit, and delete notes
- Real-time sync using Firestore
- Secure user-specific data
- Responsive UI for mobile and desktop
- Password reset support

## Tech Stack
- React
- Firebase Authentication
- Firestore
- Vite
- Tailwind CSS
